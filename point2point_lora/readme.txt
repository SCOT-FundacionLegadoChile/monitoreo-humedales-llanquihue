Para instalar librería RadioHead.h. Posee pequeñas modificaciones por lo que es necesario instalar la versión de este repositorio.
1. Mover carpeta ./RadioHead/ a carpeta ~/Arduino/libraries/
2. Verificar que librería está disponible en IDE Arduino
3. Ser feliz, salir afuera a mirar el cielo
